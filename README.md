    Gmanga is a mobile application developed using Flutter.
    The app allows users to browse manga and manhwa content, search for specific titles, and explore works by categories.

    The idea of this project is to build a simple content browsing application that demonstrates:

    The application is intended for learning purposes only and is not designed for commercial use.
